# M2 Replication Project

## Stimuli source

Object stimuli downloaded from [Bradylab](https://bradylab.ucsd.edu/stimuli.html) and modified. Same objects as in `Brady, T. F., Konkle, T.F., Gill, J., Oliva, A. and Alvarez, G.A. (2013). Visual long-term memory has the same limit on fidelity as visual working memory. Psychological Science, 24(6), 981-990.`

## Original study

Chung, Y. H., Brady, T. F., & Störmer, V. S. (2023). No Fixed Limit for Storing Simple Visual Features: Realistic Objects Provide an Efficient Scaffold for Holding Features in Mind. Psychological Science, 34(7), 784-793. https://doi.org/10.1177/09567976231171339
